# POS With Spring & Mavenüè™

This is a POS System which is developed by using Spring & Maven.

Main Functions are,
 - Manage Customers
 - Search Customer
 - Validate Customer Input Feilds
 - Manage Items
 - Search Item
 - Validate Item Input Feilds
 - Add Items to Cart
 - Place Order and Manage Discounts
 - Check Out Order
 - Prevent Focusing when Press the Tab
 
 All pages are responsive.
 
 # Built With
 Front-End
 - HTML
 - CSS
 - Java Script
 - Bootstrap
 - JQuery
 
 Back-End
 - Java/JavaEE
 - MySql
 - Spring
 - Maven
